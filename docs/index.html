Requires:
  • jQuery library 1.42+
  • jQuery UI library 1.82+ (optional)
Option Type Accepts Default Description
rotationInterval int miliseconds 5000 The time, in miliseconds, it takes before the next item is shown.
autoStart boolean true, false true If false, rotationEngine is not started on start. The rotationEngine will start when an action that starts the engine is performed, for example a mouseover event.
disableRotationEngine boolean true, false false If true, the rotationEngine will never run, even when events that is suppose to start it is fired.
shuffle boolean true, false false If true, the next item is picked on random. All items are shown once before the engine resets and starts all over again.
randomStart boolean true, false false If true, the first item is randomly picked on start. If false, the first item in the list is shown on start.
randomEffect boolean true, false false If set to true, a random effect from the randomEffects array is picked for each rotation.
randomEffects Array blind, clip, explode, fade, fold blind, clip, explode, fade, fold You can override this with your own array of effects
startIndex int 0-n 0 Which item to start with on first run. 0 is first item, n is any number between 0 and last item in the list minus one as count is zero based. This setting will be overridden if randomStart is set to true.
effect string blind, clip, explode, fade, fold, slide, none none NB: Some effects must have additional arguments to work. Currently this is the “slide” effect. Please note that aside from fade and slide, all the other effects require jQuery UI to be included. If not, it will default back to none.
effectTimer int milliseconds 1000 The time, in miliseconds, it takes for the effect to finish.
effectOptions JSON object slideBy, slideVertical, easing {} Slideby tells the engine how many pixels to move for each slide. This value must be the exact width of the content of the list element (<li>). Image width is an example of this. slideVertical is a boolean value, true slides the list vertically instead of horizontally. Easing tells the engine how to animate each slide. Defaults to swing. For additional easing, jQuery UI is required. Refer to jQuery UI easing documentation for easing names.
activeItemClass string any string alrActiveItem Class that is put on active main list item
helper jQuery selector any valid jQuery selector false If set, the main list is linked to a helper list, which can be used for thumbnails, tabs, text etc. The item order of both lists must match.
helperActiveItemClass string any string alrHelperActiveItem Class that is put on active helper list item
helperInteraction string click, mouseover mouseover Trigger event on user interaction with helper.
nextItemElement jQuery selector any valid jQuery selector false If set, interaction with the element will move the list rotation forward.
nextItemElementInteraction string click, mouseover click Trigger event on user interaction with element.
previousItemElement jQuery selector any valid jQuery selector false If set, interaction with the element will move the list rotation backwards.
previousItemElementInteraction string click, mouseover click Trigger event on user interaction with element.
itemControl JSON object rotationInterval, effect, effectTimer, effectOptions, randomEffect, randomEffects {} Lets you specify options on an per item basis. Each item is specified with listIndex_n where n is the list index, which is zero based. See below for example.
debug boolean true, false false If set to true, debug information is outputted through FireBug console (NB! requires FireBug). Warning! If set to true you might get JS errors in other browsers because of the missing FireBug plugin, use only for debugging in Firefox with FireBug.
debugLevel Array debug, info, warn, error debug, info, warn, error You can override this with your own array of debugLevels

Option example (shows how to use settings, actual setting combination might not work):

jQuery(document).ready(function($) {
    $('.myList').advListRotator({
        helper: '#myHelperId',
        rotationInterval: 2000,
        effectTimer: 500,
        effect: 'slide',
        effectOptions: {slideBy: 100, easing: 'easeInBack'},
        shuffle: true,
        activeItemClass: 'myActiveItemClass',
        randomEffect: true,
        randomEffects: new Array('fade', 'blind'),
        itemControl: {
            listIndex_0: {
                rotationInterval: 500,
                effect: 'fold'
            },
            listIndex_1: {
                effectTimer: 1500
            }
        }
    });
});
