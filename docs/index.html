Requires:
  • jQuery library 1.42+
  • jQuery UI library 1.81+ (optional)
Options:
  • rotationInterval (int) Default: 5000 The time, in miliseconds, it takes before the next item is shown.
  • effectTimer (int) Default: 1000 The time, in miliseconds, it takes for the effect to finish.
  • effect (string) Default: fade Supported effects are: blind, clip, explode, fade, fold, slide NB: Some effects must have additional arguments to work. Currently this is the "slide" effect. Please note that aside from fade and slide, all the other effects require jquery ui to be included. If not, it will default back to fade.
  • effectOptions (array) Default: {slideBy: 100} Currently supported setting is slideBy for the slide effect, it tells the engine how many pixels to move for each slide. This value must be the exact width of the content of the list element (<li>). Image width is an example of this.
  • shuffle (boolean) Default: false If true, the next item is picked on random. All items are shown once before the engine resets and starts all over again.
  • randomStart (boolean) Default: false If true, the first item is randomly picked on start. If false, the first item in the list is shown on start.
  • autoStart (boolean) Default: true If false, rotationEngine is not started on start. The rotationEngine will start when an action that starts the engine is performed, for example a mouseover event.
  • disableRotationEngine (boolean) Default: false If true, the rotationEngine will never run, even when events that is suppose to start it is fired.
  • listRotatorHelper (jQuery selector) Default: false If set, the main list is linked to a helper list, which can be used for thumbnails, tabs, text etc. The item order of both lists must match.
  • activeItemClass (string) Default: lbListRotatorItemActive Class that is put on active main list item
  • activeHelperItemClass (string) Default: lbListRotatorHelperItemActive Class that is put on active helper list item
  • helperInteraction (string) Default: mouseover Possible values are click and mouseover. Trigger event on user interaction with helper.
  • startIndex (int) Default: 0 Sets the active item on first run. 0 is first. This setting will be overriden if randomStart is set to true.
  • nextItemElement (jQuery selector) (Added in 1.2) Default: false If set, interaction with the element will move the list rotation forward.
  • previousItemElement (jQuery selector) (Added in 1.2) Defualt: false If set, interaction with the element will move the list rotation backwards.
  • nextItemElementInteraction (string) (Added in 1.2) Default: click Possible values are click and mouseover.
  • previousItemElementInteraction (string) (Added in 1.2) Default: click Possible values are click and mouseover.
Option example: 
            
            jQuery(document).ready(function($) {
                $('.myList').lbListRotator({
                    listRotatorHelper: '#myHelperId',
                    rotationInterval: 2000,
                    effectTimer: 500,
                    effect: 'slide',
                    shuffle: true,
                    activeItemClass: 'myActiveItemClass'
                });
            });
