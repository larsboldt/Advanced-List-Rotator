Requires:
  • jQuery library 1.42+
  • jQuery UI library 1.82+ (optional)

Options:

  • rotationInterval (int)
    Default: 5000
    The time, in miliseconds, it takes before the next item is shown.
  • effectTimer (int)
    Default: 1000
    The time, in miliseconds, it takes for the effect to finish.
  • effect (string)
    Default: false (none)
    Supported effects are: blind, clip, explode, fade, fold, slide
    NB: Some effects must have additional arguments to work. Currently this is the “slide” effect. Please note that aside from fade and slide, all the other effects require jQuery UI to be included. If not, it will default back to none.
  • effectOptions (JSON object)
    Default: {}
    Currently supported settings are slideBy and easing for the slide effect. Slideby tells the engine how many pixels to move for each slide. This value must be the exact width of the content of the list element (<li>). Image width is an example of this. Easing tells the engine how to animate each slide. Defaults to swing. For additional easing, jQuery UI is required. Refer to jQuery UI easing documentation for easing names.
  • shuffle (boolean)
    Default: false
    If true, the next item is picked on random. All items are shown once before the engine resets and starts all over again.
  • randomStart (boolean)
    Default: false
    If true, the first item is randomly picked on start. If false, the first item in the list is shown on start.
  • autoStart (boolean)
    Default: true
    If false, rotationEngine is not started on start. The rotationEngine will start when an action that starts the engine is performed, for example a mouseover event.
  • disableRotationEngine (boolean)
    Default: false
    If true, the rotationEngine will never run, even when events that is suppose to start it is fired.
  • helper (jQuery selector)
    Default: false
    If set, the main list is linked to a helper list, which can be used for thumbnails, tabs, text etc. The item order of both lists must match.
  • activeItemClass (string)
    Default: alrActiveItem
    Class that is put on active main list item
  • helperActiveItemClass (string)
    Default: alrHelperActiveItem
    Class that is put on active helper list item
  • helperInteraction (string)
    Default: mouseover
    Possible values are click and mouseover. Trigger event on user interaction with helper.
  • startIndex (int)
    Default: 0
    Sets the active item on first run. 0 is first. This setting will be overriden if randomStart is set to true.
  • nextItemElement (jQuery selector)
    Default: false
    If set, interaction with the element will move the list rotation forward.
  • previousItemElement (jQuery selector)
    Defualt: false
    If set, interaction with the element will move the list rotation backwards.
  • nextItemElementInteraction (string)
    Default: click
    Possible values are click and mouseover.
  • previousItemElementInteraction (string)
    Default: click
    Possible values are click and mouseover.
  • randomEffect (boolean)
    Default: false
    If set to true, a random effect from the randomEffects array is picked for each rotation.
  • randomEffects (Array)
    Default: blind, clip, explode, fade, fold
    You can override this with your own array of effects
  • debug (boolean)
    Default: false
    If set to true, debug information is outputted through FireBug console (requires FireBug)
  • debugLevel (Array)
    Default: debug, info, warn, error
    You can override this with your own array of debugLevels
  • itemControl (JSON object)
    Default: none
    Lets you specify rotationInterval, effect, effectTimer, effectOptions, randomEffect and randomEffects on an per item basis. Each item is specified with itemIndex_n where n is the item index, which is zero based. See below for example.

Option example (shows how to use settings, actual setting combination might not work):

jQuery(document).ready(function($) {
    $('.myList').advListRotator({
        helper: '#myHelperId',
        rotationInterval: 2000,
        effectTimer: 500,
        effect: 'slide',
        effectOptions: {slideBy: 100, easing: 'easeInBack'},
        shuffle: true,
        activeItemClass: 'myActiveItemClass',
        randomEffect: true,
        randomEffects: new Array('fade', 'blind'),
        itemControl: {
            itemIndex_0: {
                rotationInterval: 500,
                effect: 'fold'
            },
            itemIndex_1: {
                effectTimer: 1500
            }
        }
    });
});
